# AGENTS.md

이 파일은 `waito-spring-boot` 프로젝트에서 Codex가 작업할 때 따라야 할 사용자 지시, 개발 철학, 협업 규칙을 정리한 문서다.
새로운 요청에서 사용자가 추가 지시나 선호를 밝히면 이 파일을 함께 갱신한다.

## 사용자 지시

- 사용자가 프로젝트에서 지시한 내용과 개발 철학은 이 파일에 누적 정리한다.
- 사용자가 새 요청을 할 때마다, 해당 요청에서 새로 확인된 장기 지침이나 개발 철학이 있으면 이 파일을 업데이트한다.
- 공통적으로 앞으로도 적용되어야 할 규칙만 개발 철학에 남기고, 구체 구현 메모는 중요도가 높은 내용만 프로젝트 메모에 둔다.

## 협업 원칙

- 사용자의 요청이 구현 작업이면 제안에 머무르지 않고 가능한 범위에서 직접 코드와 설정을 수정한다.
- 변경 전에는 기존 코드 구조와 프로젝트 관례를 먼저 확인한다.
- 기존 사용자 변경사항은 되돌리지 않는다.
- 불필요한 리팩터링이나 범위 밖 변경은 피한다.
- 작업 결과는 무엇을 바꿨고 어떻게 확인했는지 간결하게 보고한다.
